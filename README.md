## 🎮 <span style="color:teal">Game Overview</span>

**DAD LEGEND** is an <span style="color:orange">**action-packed**</span> roguelike game featuring <span style="color:green">**procedurally generated**</span> challenges, <span style="color:blue">**weapon customization**</span>, and an extensive <span style="color:purple">**upgrade system**</span>. Players must survive increasingly difficult waves of enemies while building <span style="color:red">**powerful weapon combinations**</span> and managing resources through a <span style="color:gold">**dynamic shop system**</span>.

### 🌟 <span style="color:goldenrod">Core Features</span>

- **🎯 Roguelike Gameplay**: Every run is unique with <span style="color:green">**procedurally generated**</span> content
- **⚔️ Weapon Bond System**: Extensive weapon customization with <span style="color:orange">**effects**</span> and <span style="color:purple">**bonding**</span>
- **📈 Progressive Upgrading**: Character enhancement with <span style="color:blue">**multiple upgrade paths**</span>
- **🎮 Controller Support**: Full <span style="color:navy">**gamepad/controller**</span> input support
- **🏪 Strategic Shop System**: <span style="color:teal">**Resource management**</span> through an in-game store
- **👹 Epic Boss Battles**: Challenging <span style="color:crimson">**boss encounters**</span> with unique mechanics
- **🔊 Immersive Soundtrack**: Custom <span style="color:indigo">**music**</span> for atmospheric gameplay
- **⭐ Level-Up System**: Character <span style="color:darkorange">**progression**</span> with meaningful choices

## 🚀 <span style="color:orange">Getting Started</span>

### 📋 <span style="color:navy">Prerequisites</span>

- <span style="color:green">**Unity 6000.1**</span> or newer
- <span style="color:purple">**Git**</span> for version control
- A compatible <span style="color:blue">**controller**</span> (optional but recommended)

### ⚡ <span style="color:darkred">Installation</span>

1. <span style="color:blue">**Clone the repository**</span>:
```bash
git clone https://github.com/S1roha-1331/Fnal_Project_25_for_Program_Design_II.git
```

2. <span style="color:green">**Open Unity Hub**</span> and select "**Add project from disk**"

3. Navigate to the cloned folder and select the `**Finalproject**` directory

4. Wait for Unity to <span style="color:purple">**import all assets**</span> and dependencies

5. Open the <span style="color:blue">**main scene**</span> and press <span style="color:green">**Play**</span> to start the game

### 🎯 <span style="color:crimson">Controls</span>

- <span style="color:blue">**Movement**</span>: `WASD` keys or **Controller left stick**
- <span style="color:green">**Indicator**</span>: `Mouse` or **Controller right stick**
- <span style="color:purple">**Actions**</span>: `Space`/`Enter` or **Controller buttons**

### 💀 <span style="color:darkred">Roguelike Elements</span>
- **🔄 Permadeath**: When you die, you start over with <span style="color:blue">**minimal progression**</span> carried forward
- **🎲 Procedural Generation**: Each run features different <span style="color:green">**enemy spawns**</span> and <span style="color:purple">**item drops**</span>
- **⚖️ Risk vs Reward**: The longer you survive, the <span style="color:red">**stronger enemies**</span> become, but <span style="color:gold">**rewards increase**</span> accordingly

### 🔧 <span style="color:steelblue">Upgrade System</span>
Players can enhance their character through various <span style="color:indigo">**upgrade paths**</span>:
- **💥 Bullet Enhancement**: Make projectiles <span style="color:orange">**bigger**</span> and more <span style="color:red">**powerful**</span>
- **🩸 Lifesteal Ability**: <span style="color:crimson">**Recover health**</span> by dealing damage to enemies
- **🛡️ Shield System**: Gain a <span style="color:blue">**regenerable protective barrier**</span>
- **✨ Weapon Effects**: Unlock <span style="color:purple">**special abilities**</span> and combinations

### ⚔️ <span style="color:darkgoldenrod">Weapon Bond System</span>
- Weapons can be <span style="color:blue">**combined**</span> and <span style="color:green">**enhanced**</span> through a unique bonding mechanic
- Different weapon types offer <span style="color:indigo">**varied playstyles**</span> and <span style="color:crimson">**strategic options**</span>
- Discover <span style="color:gold">**powerful synergies**</span> between weapon effects
